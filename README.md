# Global News Intelligence Platform

> **Status: Active Development — Project just started, ongoing build**

A full-stack AI-powered news intelligence platform that visualizes real-time global events on an interactive world map. Click any country to receive an AI-curated news feed, category-filtered intelligence briefs, and analytics dashboards.

---

## What It Does

- **Interactive World Map** — Dark-themed Leaflet map with GeoJSON country boundaries; colored event-density overlays and animated pulse markers for breaking news
- **AI News Feed** — Click any country to get the latest news, automatically classified into 12 categories (War, Political, Geopolitical, Tech, Science, Sports, Medical, Economy, Climate, Crime, Diplomacy, General)
- **AI Intelligence Brief** — A 3–5 sentence country summary generated by a fine-tuned BART model, labeled "AI GENERATED — PyTorch NLP"
- **Time & Category Filters** — Filter by Latest / 1H / 10H / 24H / 2D / 5D / 30D and by category with visual highlights
- **Global Analytics Dashboard** — Top 10 active countries (bar chart), category distribution (donut chart), 30-day event volume timeline, heatmap overlay
- **Search** — Debounced keyword search across countries and article headlines
- **Persistent Preferences** — Last selected country, filters, and dark/light mode stored across sessions

## Architecture Overview

```
Frontend (React 18 + TypeScript)
    └── Zustand Store → WorldMap (Leaflet) → SidePanel → NewsCards
                     → Analytics Dashboard
                     → SearchBar

Backend (FastAPI)
    ├── /api/news/{country_code}   — country news feed (Redis cache-aside)
    ├── /api/analytics/*           — global & country analytics
    ├── /api/search                — full-text search
    ├── /api/ai/classify+summarize — AI inference endpoints
    └── APScheduler                — 5-min ingestion, 1-min breaking news

Data Layer
    ├── PostgreSQL 15  — news_articles, country_stats, ai_summaries
    └── Redis 7        — cache (TTL 300s per country feed)

AI Pipeline
    ├── DistilBERT     — article classification (12 categories)
    └── BART-large-CNN — country intelligence summarization

Graceful Degradation: NewsAPI → GDELT → RSS → mock seed data
                       AI model → keyword-rule fallback
                       Redis cache → direct DB query
```

---

## Key Design Goals

- Sub-1.5s country feed response via Redis cache-aside
- Decoupled AI inference pipeline behind dedicated FastAPI endpoints
- Graceful degradation at every layer — never crashes on source failure
- Stateless backend — all session state lives in Redis or the client
- Rate limiting: 100 req/min per IP on all public endpoints
